# Papillon Dog Classifier
An image classifier that identifies Papillon dogs from other breeds using transfer learning with PyTorch.

## What it does
Upload a photo of a dog and the model will predict whether it's a papillon or not.

## Tools used
- Python
- PyTorch
- Transfer Learning (ResNet18)
