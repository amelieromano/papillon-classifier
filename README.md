# Papillon Dog Classifier
An image classifier that identifies Papillon dogs from other breeds using transfer learning with PyTorch.

## What it does
Upload a photo of a dog and the model will predict whether it's a papillon or not.

## Tools used
- Python
- PyTorch
- Transfer Learning (ResNet18)

##How to run
1. Clone this repo
2. Add dog photos to data/papillon and data/other_dogs
3. Run: python3 classifier.py

##Results
The model achieved 91.7% accuracy on the test set (11/12 correct), trained on 63 dog photos using transfer learning with ResNet18.
